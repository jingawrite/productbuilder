# Product Builder

AI-first product planning workspace prototype built with Next.js.

## What is included

- Topic-first home screen that turns a product idea into a PRD draft flow
- Workspace with PRD, Features, and User Flow tabs
- Persistent left AI chat panel inspired by the provided Manifest UX
- Feature tree view and directory view
- User flow canvas with version sidebar

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
